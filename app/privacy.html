خصوصية بيانات سيرتك

هذه النسخة تعمل بنظام Local‑first: بيانات السيرة تُحفظ داخل متصفحك على هذا الجهاز، وليست مزامنة سحابية.

الحفظ والنسخ الاحتياطي

مسح بيانات المتصفح أو تغيير الجهاز قد يؤدي إلى فقدان السير. استخدم «تنزيل نسخة احتياطية» دوريًا. الملف الاحتياطي يحتوي على كل النسخ ويمكن استيراده لاحقًا.

استخدام المساعد الذكي

لا يرسل CV Studio الاسم أو البريد أو الهاتف أو العنوان أو الروابط إلى المساعد. يُرسل فقط النص المطلوب تحسينه وسياق مهني محدود: المجال، التخصص، المستوى، المسمى المستهدف والمهارات ذات الصلة.

مفاتيح API

النسخة الإنتاجية لا تطلب مفتاح API من المستخدم ولا تخزنه في LocalStorage. طلبات الذكاء الاصطناعي تمر عبر خادم الموقع مع حدود استخدام.

دقة الاقتراحات وATS

اقتراحات الذكاء الاصطناعي ونتائج ATS إرشادية وقد تخطئ. لا تعتمد أي معلومة أو مهارة أو رقم لم تؤكده بنفسك، ولا توجد نسبة تضمن القبول في وظيفة.

قبل الإطلاق التجاري

يلزم استبدال هذه الصفحة بسياسة قانونية كاملة تحدد مالك الخدمة، مزود الذكاء الاصطناعي، مدة الاحتفاظ، حقوق الحذف، والدول التي تعمل بها الخدمة.
